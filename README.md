# joes-auto-security
Windows PC security hardening project - System reconnaissance, vulnerability remediation, firewall configuration, access control, and security baseline implementation. Udacity Cybersecurity Nanodegree.


## Project Overview
Client: Joe's Auto Body  
Assignment: Secure compromised Windows PC  


---
## What I Did
### Phase 1: System Reconnaissance
- Documented hardware inventory and specifications
- Identified all installed applications
- Enumerated user accounts on system
- Assessed security services status

### Phase 2: Vulnerability Assessment
- Identified critical security gaps
- Found unauthorized accounts
- Documented disabled security controls
- Assessed compliance risks

### Phase 3: Security Hardening
- Applied 8+ security controls
- Removed unauthorized access
- Enabled core security services
- Enforced access control policies

### Phase 4: File & Data Protection
- Secured sensitive business files
- Encrypted critical data
- Configured folder permissions
- Removed non-business applications
