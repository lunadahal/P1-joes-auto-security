# joes-auto-security
Windows PC security hardening project - System reconnaissance, vulnerability remediation, firewall configuration, access control, and security baseline implementation. Udacity Cybersecurity Nanodegree.

Project Overview
Client: Joe's Auto Body - Vehicle repair services across tri-state area

Situation: Joe's business PC has been compromised by previous employees for unauthorized activities (web browsing, personal email, gaming, etc.) and is suspected to be used for malicious file transfers across the internet. The system requires immediate security hardening to industry best practices.

Assignment: Comprehensive security assessment and remediation to secure the PC for legitimate business use.

Project Type: Windows System Security & Hardening | Access Control | Vulnerability Remediation
